Finding disks for pairs of points

Once the set of pairs of points has been found, a map function computes the center of the two possible disks per each pair according to the BFE algorithm. [30] states that: ``For each such pair there are exactly two disks with radius that have those points on their circumference''. Figure 4 illustrates how to find the center of those disks.

Figure 4: Disks for [30].

After that, the complete set of candidate disks is collected and ready to be processed by the following phases of the BFE algorithm.

The implementation of the proposed method was written in Scala 2.10.6 and tested in Simba/Spark 1.6.0. The current code can be accessed at the authors' repository.
